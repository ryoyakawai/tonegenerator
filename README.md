# x-tonegenerator
## Live Demo
[https://ryoyakawai.github.io/x-tonegenerator/](https://ryoyakawai.github.io/x-tonegenerator/)

## What is this?
This is a Polymer Element of "PCM Tone Generator" powered by Web Audio.

## Installation
### Get component
#### bower
```shell
$ bower install x-tonegenerator;
```
